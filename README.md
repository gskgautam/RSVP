# RSVP-X Dataset

RSVP-X is a curated benchmark dataset designed for evaluating Large Language Models (LLMs) under the **Helpfulness, Harmlessness, and Honesty (HHH)** paradigm.

---

## 📦 Dataset Overview

- **Total Samples:** 9,228  
- **Total Domains:** 106  
- **Language:** English  

RSVP-X captures alignment across four key dimensions:
- 🧠 Reasoning (14 domains)  
- 🛡️ Safety (14 domains)  
- ⚖️ Value (56 domains)  
- 🌍 Pluralistic (22 domains)  

> The dataset is constructed to enable **multi-dimensional alignment evaluation**, rather than isolated benchmarking. :contentReference[oaicite:0]{index=0}
